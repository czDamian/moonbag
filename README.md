# Moon Bag

Moon Bag is a modern, community-driven DAO dashboard for managing and viewing treasury assets, token holdings, and more. Built with Next.js and Tailwind CSS, it features a responsive UI, wallet integration, and interactive components for a seamless DeFi experience.

## Features

- **Treasury Overview:**  
  View DAO treasury address, total value, and asset breakdown.

- **Token Holdings:**  
  Paginated and filterable list of all tokens held by the treasury, with live search and social links.

- **Send to Treasury:**  
  Easily send tokens to the DAO treasury with a guided form and token selector.

- **Wallet Integration:**  
  Connect your Solana wallet to interact with the DAO.

- **Responsive Design:**  
  Fully responsive layout for desktop and mobile, including a sidebar and mobile navigation.

- **Animated Featured Tokens:**  
  Highlighted tokens with animated backgrounds for special assets.

- **Social & Community Links:**  
  Quick access to DAO social channels and community resources.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Solana Wallet Adapter](https://github.com/solana-labs/wallet-adapter)
- [React Icons](https://react-icons.github.io/react-icons/)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/czDamian/moonbag.git
   cd moonbag
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in your browser:**  
   Visit [http://localhost:3000](http://localhost:3000)
